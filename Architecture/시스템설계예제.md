## 질문  
새로운 포스트를 올리고 다른 친구의 뉴스 피드를 볼 수 있는 기능을 가진 시스템을 설계하라. 

<br />
<br /> 

## 설계   
이 문제는 두 가지 처리 flow로 나눠 생각해 볼 수 있다.   

#### 피드 발행 
사용자가 포스트를 올리면 관련된 데이터가 캐시/데이터베이스에 기록되고,   
해당 사용자의 친구 뉴스 피드에 뜨게 된다.


#### 피드 생성 
어떤 사용자의 뉴스 피드는 해당 사용자 친구들의 포스트를 시간 역군으로 정렬하여 만든다.
